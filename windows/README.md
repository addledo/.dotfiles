### To configure opener for Windows

1. Set environment variable `YAZI_CONFIG_HOME` to the directory the yazi.toml file will be in. E.g. `C:\Users\addledo\.config\yazi`
2. Add the file to that directory

### To configure file type function for Windows
1. Install git
2. Set the environment variable `YAZI_FILE_ONE` to the location of the `file.exe` executable that was installed with git.
